# 🛍️ Store-Landing-Page

A modern e-commerce store built with React.js. The application fetches product data from an external API and displays products in a clean, user-friendly interface.

## 🚀 Features

- Product listing using API
- Clean and modern UI
- Product cards with image, title, and price
- React Components
- Dynamic data rendering
- Fast and lightweight

## 🛠️ Built With

- React.js
- JavaScript (ES6)
- Tailwind CSS
- CSS3
- HTML5
- Fake Store API

## 📂 Project Structure

```
src/
├── components/
├── pages/
├── App.jsx
├── main.jsx
```
